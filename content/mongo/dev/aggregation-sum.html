Note
 count 数据类型不全是 int32，还可能是 string, double, 或 undefined。

目的

计算中 count 的总数，且需要忽略错误的字段。

答案
> db.data.aggregate([
  {
    "$unwind": "$turnstile"
  },
  {
    "$group": {
      "_id": 0,
      "sum": {
        "$sum": {
          "$convert": {
            "input": "$turnstile.count",
            "to": "int",
            "onError": 0,
            "onNull": 0
          }
        }
      }
    }
  }
])

{ "_id" : 0, "sum" : 281245 }
